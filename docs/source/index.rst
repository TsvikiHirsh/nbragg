nbragg Documentation

nbragg is a package designed for fitting neutron Bragg edge data using NCrystal cross-sections.

Features
--------

- Flexible Cross-Section Calculations
- Built-In Tools for Response and Background Functions
- LMFit Integration
- Pythonic API
- Plotting Utilities
- Bragg Edge Analysis
